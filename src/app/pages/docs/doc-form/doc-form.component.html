sql-form

Container for table driven form for adding, editing, and deleting records from a table.

Parameter Usage/Default Description [table] [table]="'psp_survey_template'" Base table the form maintains. Do not forget the single ticks inside the double quotes when using. [card] [card]="'Y'" (default 'Y') If 'Y' form will be inside a bootstrap card with the title in the header. If you want a closable card use sql-card. [id] [id]="id" Primary key if you are editing an existing record. '0' inserts a new record. [class] [class]="'m-3 p-3'" (default m-2 p-2) Classes applied to the form container. [style] [table]="'background-color: lightgrey;'" (default blank) Style applied to the form container. [bs_row] [bs_row]="'Y'" (default 'Y') If 'Y' a new bootstrap row is added. [bs_col] [bs_col]="'col-xl-6 col-sm-12'" (default blank) If not blank, classes applied to the column containing the component. [handler] [handler]="'post_template_form'" (default blank) If not blank, classes applied to the column containing the component.
